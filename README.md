# Проект «react-burger»

## Описание:
Создай свой космический бургер, только здесь самые вкусные булки и начинки. 

## Стек:
1) HTML
2) CSS
3) JS
4) React

## Технологии:
1) Флексбокс-вёрстка
2) Построение сеток с использованием технологии CSS Grid Layout
3) Работа с GIT
4) Работа с Webpack
5) Работа с библиотекой React
6) Работа с React-DND
7) Работа с API
